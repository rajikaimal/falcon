# falcon

A static site generator running on Bun

To install dependencies:

```bash
bun install
```

Dev mode:

```bash
bun index.ts --dev
```

Build:

```bash
bun index.ts --build
```

## Routing

- File based routing (inspired by Next.js).
- Every page is under `/pages` directory
- Live reload - ⏳
- Top level pages routing - ✅
- CSS inline styles - ✅
- CSS modules - ⏳
- Layouts - ⏳
